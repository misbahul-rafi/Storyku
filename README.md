# Storyku - Misbahul Rafi
### Introduction
Storyku adalah aplikasi tempat menyimpan cerita yang dapat memudahkan user untuk menggambarkan imajinasi penulis dalam bentuk tulisan. Dengan adanya Storyku user akan dimudahkan dalam mengelola hasil tulisan secara online

### Feature
- Menampilkan Daftar Story
- Menampilkakn Daftar Chapter
- Menambahkan Story
- Menambahkan Chapter

### Libraries
- React JS
- Express JS
- PostgresSQL
- etc.
